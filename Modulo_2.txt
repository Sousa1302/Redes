Roteamento Estático e Dinâmico:
-> A diferença entre o roteamento estático e o dinâmico é que as rotas estáticas são inseridas manualmente na tabela de roteamento pelos administradores de rede. Já o roteamento dinâmico utiliza protocolos de roteamentos que ajustam automaticamente as rotas

Roteamento Estático: Melhor para redes pequenas, nao precisa de grande processamento e nao ocupa largura de banda 
Roteamento Dinâmico: Melhor para redes maiores, está sempre a atualizar as suas rotas

Next hop: É um termo de routing que se refere ao endereço do router mais proximo. É utilizado num conjunto de routers conectados entre si e é o próximo endereço possivel para o qual se pode enviar pacotes.

Firmware: É um tipo de software que é armazenado em um dispositivo eletrônico e que controla o funcionamento deste dispositivo. Ele é responsável por gerenciar as funções básicas do hardware, como o processamento de dados, a comunicação com outros dispositivos e a entrada e saída de dados.

CMOS : É uma pequena quantidade de memória em uma placa-mãe do computador que armazena as configurações da BIOS (Basic Input/Output System — Sistema Básico de Entrada/Saída).

POST : Quando o computador é inicializado a BIOS executa o POST, este é um programa de diagnostico embutido que verifica o hardware, assegurando o funcionamento correto. O POST é executado muito rapidamente, e normalmente não é notado.

Kernel : É o núcleo do sistema operativo, coração do computador. Trata-se do software que reside na memória e instrui a CPU sobre o que fazer.

Sistema operativo : É um software, ou conjunto de softwares, cuja função é administrar e gerenciar os recursos de um sistema, desde componentes de hardware e sistemas de ficheiros a programas de terceiros, estabelecendo a interface entre o computador e o utilizador. (kernel)

Interpretador de comandos (Shell) : É o programa responsável em interpretar as instruções enviadas pelo utilizador e os seus programas ao sistema operativo (o kernel). Ele é quem executa comandos lidos do dispositivo de entrada padrão (teclado) ou de um ficheiro executável.

Sistema de Ficheiros : É um conjunto de regras e estruturas usadas para organizar, gerenciar e armazenar informações em HDDs, módulos SSDs, CDs, DVDs, pen drives e cartões de memória. São essas regras que determinam como os dados serão gravados no dispositivo de armazenamento.
